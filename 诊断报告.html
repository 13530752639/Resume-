💡 技术细节：

npm install 在安装依赖包时，会同时在 node_modules/.bin/ 目录下创建符号链接， 指向各包中的可执行文件。例如 vite 包的可执行文件会被链接到 node_modules/.bin/vite。 当这个目录缺失时，npm scripts 中的所有命令（如 vite、tsc 等）都无法执行。

🔧 推荐修复步骤

步骤 1：清理现有安装

删除不完整的 node_modules 目录和锁文件：

rm -rf node_modules package-lock.json
步骤 2：切换 npm 镜像源（可选）

如果网络持续不稳定，可切换到官方源或其他镜像：

步骤 3：重新安装依赖

使用以下命令重新安装所有依赖：
