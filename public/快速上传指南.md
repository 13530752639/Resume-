# 🚀 快速上传指南

## 📤 上传视频作品(3步完成)

### 第1步: 准备文件
- 视频文件: `my-documentary.mp4` (MP4格式)
- 封面图片: `my-documentary-cover.jpg` (16:9比例)

### 第2步: 放置文件
```bash
# 视频文件
public/videos/documentary/my-documentary.mp4

# 封面图片
public/covers/video-works/documentary/my-documentary-cover.jpg
```

### 第3步: 更新数据
打开 `src/data/categories.json`,在对应分类添加:

```json
{
  "id": "doc-003",
  "title": "我的纪录片",
  "titleEn": "My Documentary",
  "category": "纪录片",
  "thumbnail": "/covers/video-works/documentary/my-documentary-cover.jpg",
  "videoUrl": "/videos/documentary/my-documentary.mp4",
  "duration": "20:00",
  "description": "纪录片简介"
}
```

## 📸 上传摄影作品(2步完成)

### 第1步: 准备并放置文件
```bash
# 图片文件
public/images/portrait/my-portrait.jpg
```

### 第2步: 更新数据
打开 `src/data/categories.json`,在对应分类添加:

```json
{
  "id": "portrait-003",
  "title": "人物肖像",
  "titleEn": "Portrait",
  "category": "人像摄影",
  "thumbnail": "/images/portrait/my-portrait.jpg",
  "description": "作品说明"
}
```

## ⚠️ 注意事项

1. **文件命名**: 使用英文和数字,避免空格
2. **ID唯一**: 每个作品ID不能重复
3. **路径正确**: 使用 `/` 开头的相对路径
4. **保存刷新**: 修改后保存文件,浏览器自动刷新

## 🎯 完整示例

### 上传一部纪录片"城市之光"

**步骤1**: 放置文件
```
public/videos/documentary/city-lights.mp4
public/covers/video-works/documentary/city-lights-cover.jpg
```

**步骤2**: 编辑 `src/data/categories.json`
```json
{
  "videoWorks": {
    "documentary": [
      {
        "id": "doc-003",
        "title": "城市之光",
        "titleEn": "City Lights",
        "category": "纪录片",
        "thumbnail": "/covers/video-works/documentary/city-lights-cover.jpg",
        "videoUrl": "/videos/documentary/city-lights.mp4",
        "duration": "25:00",
        "description": "记录城市夜晚的璀璨灯光与人文故事"
      }
    ]
  }
}
```

**步骤3**: 保存文件,刷新浏览器,完成! ✅
